# Data Analysis Projects

This repository contains two  separate data analysis projects examining different datasets.
this was done and created to do 2 things 
- to learn and try out different pull request 
-data analysis practice 

## Getting Started

Each project folder contains its own README and Final Reports with specific instructions and details about the analysis. The analyses are conducted using Jupyter notebooks with Python.
you can fork it or clone it for your self
### Prerequisites
- Python 3.x
- Jupyter Notebook
- Required Python packages (specified in each project's README)
